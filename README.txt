Python 3.7.3


Данный парсер создан для отлеживания информации с определенного сайта и отправления данных в телеграм канал

Для корректной работы парсера необходимо создать файл key.py и поместить туда токен вашего бота, например - token = '000000:XXXXXX'
Так же необходимо создать файл answer.json, в который будет помещаться json-ответ от telegram

Чтобы бот мог отправлять запросы пользователя, используй webhook https://core.telegram.org/bots/api#setwebhook
